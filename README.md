# 影视分析可视化系统 python239

## 项目概述

本项目是一个基于Django框架开发的影视分析可视化系统。系统实现前后端分离，具有完善的功能和清晰的代码结构，提供用户登录注册、影视信息浏览、搜索、评分、点赞、收藏等功能，并通过数据分析、词云分析等模块，让用户能直观地看到数据趋势。

## 技术栈

- 前端：基于Django框架
- 后端：Django框架
- 推荐算法：协同过滤算法

## 功能模块

### 前台用户角色功能

- 登录与注册
- 浏览影视信息
- 浏览影视标签
- 搜索影视信息
- 基于协同过滤推荐算法的影视推荐（用户相似度与物品相似度）
- 影视评分
- 影视点赞与收藏
- 影视数据分析
- 影视词云分析
- 影视数据分析大屏看板系统

### 管理员角色功能

- 管理员登录
- 用户管理
- 标签管理
- 影视管理
- 评分管理
- 点赞管理
- 用户角色权限管理

## 系统角色

- 普通用户：使用系统的各项功能
- 管理员：负责系统的维护与管理
